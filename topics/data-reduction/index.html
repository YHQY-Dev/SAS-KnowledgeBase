数据还原与校正（Data reduction）

From 2D detector image to corrected $I(q)$ — Pauw 2013 guided reading

本导读以 Pauw（2013）Everything SAXS: small-angle scattering pattern collection and correction（DOI: 10.1088/0953-8984/25/38/383201）为主线，系统梳理从原始探测器图像到可分析 $I(q)$ 的全部校正步骤、误差传播与顺序；并以 Benecke 等（2014）DPDAK 包为例说明大型设施上的批处理还原实践。Pauw 的核心论断：分析质量的上限由数据质量决定——再精巧的拟合也无法挽救未校正或误校正的原始图样。

还原（reduction）与绝对标定（AU）的关系：前者把探测器计数变成物理一致的相对 $I(q)$（已扣背景、归一透射/时间/厚度、正确 $q$ 轴与误差）；后者再乘标定因子得到 $\mathrm{d}\Sigma/\mathrm{d}\Omega$（见 绝对标定专题）。许多「拟合发散」「$R_g$ 不对」的根因其实在还原链前段——背景、$q$ 刻度、bin 不确定度——而非模型本身。

问题是什么

小角散射实验测得的是探测器上的计数或强度分布，而非样品的微分截面 $\mathrm{d}\Sigma/\mathrm{d}\Omega$。从像素到 $I(q)$ 须经过一系列几何、探测器、样品与仪器效应的校正。若跳过或顺序错误：

还原错误在分析阶段的表现往往是「隐蔽的」：$R_g$ 可能仍能从弯曲的 Guinier「拟合」出来，但 $I(0)$、Porod 尾、条纹位置会系统性偏移。Jeffries 2021 强调现代 SAS 应从可重复还原起步；Pauw 2013 附录 A 给出测量前/后检查清单，适合作为 beamtime 标准操作程序（SOP）。

  • $q$ 刻度偏差 → $R_g$、条纹位置系统性错误；
  • 背景未扣或扣错 → 低 $q$ 抬高、Porod 尾失真；
  • 透射/厚度未归一 → 不同样品强度不可比；
  • 环向积分 bin 误差低估 → 拟合 $\chi^2$ 虚假偏小（Pauw 2014 勘误强调须用均值标准误而非 bin 内样本标准差）；
  • 球面角/偏振忽略 → 高 $q$ 或宽角区系统偏差（通常 $<2\%$，同步辐射偏振显著时须校正）。

Pauw 将 SAS 实验分为三步：采集 → 校正还原 → 分析。本章聚焦第二步；绝对标定（AU）见 绝对标定专题 与 学习章 05。

还原链的「顺序敏感」常被低估：背景应在相同样品归一化之后扣除，还是之前？透射校正是否已含在 $I_0$ 监测比中？不同软件默认不同，混用可导致 5–10% 系统差（Pauw §3.5）。发表与归档时应记录：软件名、版本、校正列表与顺序——Trewhella 2017 与 NeXus 类元数据格式的要求与此一致。

文献主线

Pauw 2013：「Everything SAXS」的结构

全文 24 页，按以下逻辑展开（§3 为核心）：

  1. §1 引言： SAS 的相位问题、形状/多分散/堆积三选二困境；仪器亮度从 $10^3$–$10^4$ ph/s（1930s）增至 $10^{11}$–$10^{14}$ ph/s（现代同步辐射）；探测器从胶片 → 成像板 → 气体线阵 → CCD → 光子计数 PAD（Pilatus/Eiger/Medipix）。
  2. §2 采集： 针孔 / 狭缝 / Bonse–Hart 三类仪器；真空样品室 vs 空气路径对弱散射样品的影响；半透射束阻或 PIN 二极管监测 $I_0$ flux。
  3. §3 还原与校正： 逐条列出校正代号（MK、PO、TR、FL、TI、TH、AU、BG、SP、SA、MS、SM、BIN 等），给出公式与不确定度传播；§3.5 给出标准样品校正顺序；附录为测量检查清单。
  4. §4–5： 校正后数据如何支撑拟合；强调 desmearing 数据不如 smearing 模型（与 Pedersen 1995 一致）。

Pauw 的「代号语言」价值在于跨软件、跨台站沟通：方法节写「TR、FL、BG、BIN，按 §3.5 顺序」比只写软件按钮名更可审稿。§3.4 对每一步给出像素级 $\sigma$ 如何传播到 $I(q)$ 的方差——这是 Pauw 2014 勘误的上下文：bin 阶段若误用样本标准差，会连锁导致 Guinier、IFT 权重错误。

Pauw 2014 勘误：bin 不确定度

原文 §3.4.10 式 (35) 误用 bin 内样本标准差作为均值不确定度；正确做法取传播不确定度与均值标准误（SEM）的较大者，并可设相对不确定度下限 1%（即使校正再严也难以更准）。这对生物 SAXS 低 $q$ 区 Guinier 拟合权重至关重要。

Benecke 2014（DPDAK）：设施级批处理还原

DESY 开发的 DPDAK 面向海量 2D 图（SAXS、GISAXS、时间分辨），把 Pauw 类校正封装为可定制流水线：坏点掩膜、几何标定、通量/透射归一、背景扣除、1D 积分与可选 GISAXS 专用路径。与交互式 FIT2D / Nika 相比，DPDAK 强调：

  • 可重复脚本化： 同一 beamtime 数百帧用同一校正链，避免「手点」不一致；
  • 与设施几何绑定： 像素尺寸、$L$、束心一次标定、批量应用；
  • 局限： 不替代用户对背景匹配、各向异性是否该积分、smearing 策略的判断——输出仍须 QC（Guinier 线性、overlap 区 merge 等）。

选用 DPDAK 或台站自研软件时，应把 Pauw 代号列表当作配置检查表，确认未跳过 TR/FL/BG 等关键步。

径向积分 vs 扇区积分

多距离 / 多曝光拼接（Cookson 2006；Pauw §3.4）

现代 SAXS 常在多个样品–探测器距离 $L$ 或多曝光时间上拼接 $q$ 范围。还原要点：

  • 每个 $L$ 独立 $q$ 标定与束心；overlap 区强度应平滑至 $<5\%$；
  • CCD 动态范围有限时，短曝光补高 $q$、长曝光补低 $q$（或分区 mask 拼接）；
  • merge 时须统一 TR、FL、TI 归一化后再拼接，勿对未归一计数直接 splice；
  • 变能量（ASAXS）时 $I_0$/$I_t$ 响应可能变，须二级环状标样逐能量归一（Cookson 2006）。

关键公式与结论

探测器校正（§3.3）

依探测器类型不同，可能包括：暗电流/偏移（OF）、增益/非线性（NL）、坏点掩膜（MK）、几何畸变（GD）、点扩散函数（PSF）。光子计数 PAD（Pilatus/Eiger/Medipix）通常仅需暗电流与坏点；CCD 须额外非线性与荧光校正。掩膜不改变不确定度；非线性校正会改变 $\sigma_{r,j}$。

计数统计： Poisson 区 $\sigma \approx \sqrt{N}$；经除法归一（TR、FL）后须误差传播。Pauw 给出逐像素 $\sigma_{r,j}$ 经整条校正链的传播公式（§3.4）——拟合权重应基于传播后的 $\sigma(q)$，而非原始计数。

$q$ 刻度与几何（MK 之后、BIN 之前）

散射矢量（本站 $q$）：

$$q = \frac{4\pi\sin\theta}{\lambda}.$$

银二十二酸、硬脂酸或光栅峰定 $L$ 与束心；改 $L$、$\lambda$、探测器倾斜后必须重标。Cookson 2006 用内置二氧化硅环自动拟合束心；多 $L$ merge 时 overlap 区应平滑至 $<5\%$ 台阶，否则 $q$ 或归一有误。

透射与通量（TR、FL）

透射因子（离子室法，同步辐射常见）：

$$T_{\mathrm{r}}=\frac{I_{\mathrm{d},1}I_{\mathrm{u},0}}{I_{\mathrm{u},1}I_{\mathrm{d},0}}.$$

台式机常用半透射束阻或下游强散射体间接测 $T_r = I_1/I_0$。校正：

$$I_{j,\mathrm{cor}}=\frac{I_j}{T_{\mathrm{r}} f_{\mathrm{s}}}.$$

薄样品 $T_r = \exp(-\mu d)$；$T$ 误差直接进强度误差预算。

时间与厚度（TI、TH）

$$I_{j,\mathrm{cor}}=\frac{I_j}{t_{\mathrm{s}} d_{\mathrm{s}}}.$$

使不同曝光与光程的测量可比；不确定度须传播 $\sigma(t_s)$、$\sigma(d_s)$。

样品自吸收（SA）

厚样品或高 $\mu$ 材料：散射路径上的吸收使高角强度偏低。Pauw 给出校正因子（取决于几何与 $\mu$）；金属、颗粒悬浊液、未对比匹配的中子样品须检查。判断：变厚度后高 $q$ 形状是否一致。

背景扣除（BG）

$$I_{j,\mathrm{cor}}=I_j-I_{j,\mathrm{b}}, \quad \sigma_{\mathrm{r},j,\mathrm{cor}}=\sqrt{\sigma_{\mathrm{r},j}^{2}+\sigma_{\mathrm{r},j,\mathrm{b}}^{2}}.$$

背景测量须与样品测量在池型、位置、曝光、校正链上尽可能一致（Brület 等，Pauw 引文）。样品/背景信噪比大时，可适当缩短背景曝光以优化总统计（Pauw §3.4.5）。

球面角校正（SP）

平面探测器像素对应立体角不等；归一化校正（Pauw 式 27）：

$$I_{j,\mathrm{cor}}=I_j\frac{L_{\mathrm{P}}^{3}}{L_{0}^{3}}.$$

$2\theta < 5°$ 时通常 $<1\%$；宽角 WAXS 拼接时不可忽略。

偏振（PO）

同步辐射 $P_i \approx 0.95$ 时，$2\theta \approx 5°$ 处校正约 1%。未偏振光取 $P_i=0.5$。公式见 Pauw 式 (18)–(19)（Hura 等）。

环向积分 / binning（BIN）

各向同性样品：按 $q$ 分 bin，计算 $\overline{q}$ 与 $I_{\mathrm{qbin}}(\overline{q})$。不确定度（Pauw 2014 勘误后）：

$$\sigma_{r,q\mathrm{bin}}=\max\left\{\frac{1}{N_{q\mathrm{bin}}}\sqrt{\sum_{q\in[q_n,q_{n+1}]}\sigma_{r,j}^{2}},\;\frac{1}{\sqrt{N_{q\mathrm{bin}}}}\sqrt{\frac{1}{N_{q\mathrm{bin}}-1}\sum_{q\in[q_n,q_{n+1}]}(I_j-I_{q\mathrm{bin}})^{2}}\right\}.$$

bin 间距可选均匀、对数或自适应（Ilavsky 2012；Cervellino 2006）。低 $q$ 建议对数 bin 以保留 Guinier 区点数。

bin 方案优点缺点适用
线性 $q$高 $q$ 分辨均匀低 $q$ 点稀疏Porod 尾、宽 $q$ 模型拟合
对数 $q$低 $q$ 点多高 $q$ 点疏生物 Guinier、GNOM 输入
自适应信噪加权实现复杂弱散射、宽动态范围

误差传播与拟合权重（Pauw 2014）

还原后每点 $\sigma(q)$ 应综合：计数统计、背景扣除、透射/厚度不确定度、bin 内 SEM。若仅用 $\sqrt{N}$ 而忽略系统项，$\chi^2 \ll 1$ 导致过拟合。Pauw 建议相对不确定度下限 1%，反映未建模的系统误差（狭缝 smearing、池壁散射等）。

仪器 smearing（SM）与多重散射（MS）

贡献来源：$\Delta\lambda/\lambda$、束发散、束斑尺寸、样品厚度、探测器 PSF。针孔 + 晶体单色通常 $\Delta\lambda/\lambda < 10^{-3}$，smearing 由准直主导；Bonse–Hart 须显式处理波长展宽。Pauw 强烈建议：拟合时对模型做 smearing，而非对数据 desmear——desmear 放大噪声甚至引入伪特征（与 Pedersen 1995 一致）。Kratky 狭缝数据若已 desmear，须报告算法与 $q$ 范围；生物 pinhole 数据 smearing 常可忽略，但 USAXS–SAXS 拼接仍须注意。

生物 SAXS 还原要点（Jeffries 2021；Trewhella 2017）

生物样品还原的「特殊之处」不在公式，而在背景与样品 QC：

  • 缓冲液匹配： 样品+缓冲液与纯缓冲液须同池型、同光路、同曝光策略；透析液作 blank 是常用最佳实践。
  • 浓度系列： 还原后先检查 $I(q)/c$ 在低 $q$ 是否重合，再进入 Guinier/IFT——还原错误与聚集在此一并暴露。
  • SEC-SAXS： 峰内积分相当于在线稀释；须对齐 UV 与 SAXS 时间轴，否则 $c$ 与 $I(0)$ 不对应。
  • 辐射损伤： 多子曝光或流动池；对比帧间 $I(0)$ 漂移 $>3\%$ 须丢弃或降通量。

绝对标定（AU）对生物 $M$ 验证是还原链的延伸，见 标定专题；但即使不做 AU，相对还原仍须严格，否则 $R_g$、Kratky 形态亦不可信。

各向异性与 2D 数据保留

还原不一定是「2D → 1D」。纤维、剪切、液晶、GISAXS 等应保留 $(q,\psi)$ 或 sector 曲线（Pauw §3.4.9）。全环平均前的自检：2D 图是否圆对称？azimuthal 剖面是否平坦？若否，勿强行 BIN 成单一 $I(q)$。

实践要点与坑

  • 背景不匹配 → 如何判断： 样品+缓冲液与纯缓冲液 $I(q)$ 在低 $q$ 不能相减至近零；$R_g$ 偏大、$p(r)$ 尾部抬高。检查池型、 meniscus 位置、是否同一光路。
  • $q$ 标定漂移 → 如何判断： 标准样峰位随时间移动；多距离 merge 在 overlap 区台阶 $>5\%$。每次改 $L$ 或能量后重标。
  • bin 不确定度低估 → 如何判断： $\chi^2 \ll 1$ 且残差无结构；Guinier 拟合权重失真。用 Pauw 2014 勘误公式；设 1% 相对下限。
  • 透射未测 → 如何判断： 厚样品高 $q$ 相对强度系统性偏低；不同厚度同一样品曲线不能归一重合。$T$ 应独立测量，勿从散射反推。
  • 先扣背景还是先除 $T$ → 如何判断： 软件顺序须与文献/台站一致；混用会导致 5–10% 系统差。记录软件默认顺序（Pauw §3.5 给出推荐序）。
  • 对 Kratky 数据 desmear → 如何判断： 低 $q$ 出现非物理振荡；$R_g$ 偏大 10–30%。改 smeared 模型拟合或针孔重测。
  • 宇宙线/坏点未掩 → 如何判断： $I(q)$ 上孤立尖峰；环向平均后仍残留 zingers。还原前掩膜或中值滤波。
  • 多重散射 → 如何判断： $T < 0.37$ 且浓溶液；变厚度或 $\lambda$ 后 $I(q)$ 形状变。稀释样品或理论/蒙特卡罗校正。
  • 2D 取向被平均掉 → 如何判断： 2D 图有明显扇区差异但 1D 曲线「过于光滑」；纤维样品 $R_g$ 与显微镜不符。改扇区积分或 GISAXS 专题。
  • 多 $L$ / 多能量 merge → 如何判断： overlap 台阶、束心不一致、$I_0$ 归一未随能量更新（Cookson 2006）。每段独立 $q$ 标定后 merge，或固定二级环强度曲线。
  • 元数据缺失 → 如何判断： 无法复现还原即无法审稿；须存档 $\lambda$、$L$、曝光、池型、软件版本、校正列表（Jeffries 2021）。
  • 还原后直接建模 → 如何判断： 跳过 Guinier/$p(r)$ QC 时，建模 $\chi^2$ 无诊断价值。先验证低 $q$ 线性、$M$（若绝对标定）、浓度系列。
