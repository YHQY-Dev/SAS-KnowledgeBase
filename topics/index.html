全部专题（12）

绝对标定 已就绪 实验室与同步辐射 SAXS 的绝对强度、标样选择与 NIST SRM 3600；如何把 $I(q)$ 放到 $\mathrm{d}\Sigma/\mathrm{d}\Omega$ 尺度。 数据还原 已就绪 2D 图样→$I(q)$、背景扣除、透射率与误差传播；校正链条的系统梳理。 IFT / GNOM 已就绪 从 $I(q)$ 到 $p(r)$ 的间接傅里叶变换、正则化与解的唯一性直觉。 生物 SAS 已就绪 溶液中生物大分子的 SAXS：样品质量、Guinier、$p(r)$、低分辨建模与发表规范。 ATSAS 工具链 已就绪 ATSAS 程序族导读：PRIMUS、GNOM、DAMMIF、CRYSOL 等在典型流程中的位置。 SANS 衬度变化 已就绪 氘代与匹配点；多组分复合物的衬度设计与 Stuhrmann 分析入门。 GISAXS 已就绪 掠入射几何、2D 图样、DWBA 与薄膜/表面纳米结构的散射解释。 ASAXS 已就绪 反常小角散射：元素分辨与近边能量扫描在纳米结构中的应用。 磁性 SANS 已就绪 磁散射衬度、自旋与磁纳米结构的 SANS 信号解读。 聚合物 / 嵌段共聚物 已就绪 链构象、微相分离与 BCP 薄膜的 SAS 特征；与 GISAXS 的衔接。 USAXS 已就绪 超小角区仪器、狭缝效应与多尺度拼接；与 pinhole SAXS 的绝对标定联系。 样品环境 已就绪 原位池、温控、流动与 SEC-SAS 等环境装置对数据质量的影响。

怎么选专题

  • 数据只有任意单位、无法算分子量或体积分数 → 从 绝对标定 开始。
  • 薄膜或表面纳米结构、需要 2D 散射图 → GISAXS。
  • 蛋白质/核酸溶液、结构生物学问题 → 生物 SAS。
  • 还不熟悉 $q$、$P(q)$、$S(q)$ → 先读 学习路径 02，再回专题。
