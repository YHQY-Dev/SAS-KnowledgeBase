# SAS 知识站

面向小角散射（SAXS / SANS / GISAXS 等）的中文教程站。

## 内容

| 目录 | 说明 |
|------|------|
| `learn/` | 学习路径：从基础到数据还原、标定、Guinier / IFT、模型与应用 |
| `topics/` | 专题手册：绝对标定、BioSAS、GISAXS、衬度变化等 |
| `glossary/` | 中英术语表（文中可悬停、可跳转） |
| `catalog/` | 核心文献题录（题名、作者、年份、DOI / ISBN），不含全文 |

正文以中文为主，专业术语保留英文。站内搜索：顶栏「搜索」或 Ctrl+K。

## 说明

- 纯静态站点，无构建步骤
- 教程为学习向改写与导读；引用具体结论请回查原文
