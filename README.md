# SAS 知识站

面向小角散射（SAXS / SANS / GISAXS 等）的中文教程站：一条从零开始的学习路径，一套按主题查阅的专题手册，可悬停、可跳转的中英术语表，以及可按 DOI 打开原文的文献题录。

## 本地预览

用任意静态服务器打开本仓库根目录，例如：

```bash
npx --yes serve .
```

也可直接用浏览器打开 `index.html`（`file://` 可用）。

## 内容结构

- `learn/` 学习路径：从基础概念到数据还原、标定、Guinier / IFT、模型与应用岔路
- `topics/` 专题手册：按实验问题深入（绝对标定、BioSAS、GISAXS、衬度变化等）
- `glossary/` 术语表：中英双语词条；正文虚线术语可悬停与跳转
- `catalog/` 文献目录：核心文献题录（题名、作者、年份、DOI / ISBN），不含全文

站内搜索：顶栏「搜索」或 Ctrl+K。
