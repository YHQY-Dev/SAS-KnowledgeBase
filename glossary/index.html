怎么用这套术语

  • 读教程时：遇到虚线术语 → 悬停确认英文全称 → 需要细节再点进词条。
  • 查文献时：先在左侧找缩写（如 $R_g$、GISAXS、GNOM），再跟到相关学习章或专题。
  • 写报告时：核对单位与符号约定（$q$ vs $s$、绝对强度 $\mathrm{cm}^{-1}$、衬度匹配用语等）。

按主题进入（可点）

每组只列最常用入口；其余词条请用左侧侧栏浏览。

探针与实验变体

SAXS 小角 X 射线散射：电子密度起伏、实验室与同步辐射的通用探针。 SANS 小角中子散射：核 SLD、H/D 衬度操控；与磁散射边界见磁性 SANS。 GISAXS 掠入射几何；配套 DWBA、Yoneda、临界角。 USAXS 超小角 / 微米尺度；常遇 Bonse–Hart 与 desmearing。 ASAXS 反常小角散射；依赖 反常色散 $f'$、$f''$。 WAXS 广角区：晶格与分子堆积；常与 SAXS/USAXS 跨尺度联用。

核心物理量

散射矢量 $q$ 动量转移与实空间尺度窗口；$d\sim 2\pi/q$。 散射长度密度 SLD / 电子密度；强度量级的来源。 衬度 $\Delta\rho$；延伸到 衬度匹配 与 衬度变化。 形式因子 $P(q)$ 单粒子形状与尺寸；与 结构因子 $S(q)$ 成对出现。 相干散射 结构信息的载体；对照 不相干本底。 体积分数 $\phi$ 绝对强度下浓度/相含量解释的关键参数。

读谱与初级分析

Guinier 分析 低 $q$ 求 $R_g$ 与 $I(0)$。 Porod 定律 高 $q$ 界面标度；相关 Porod 不变量。 Kratky 图 紧凑 / 柔性 / 无序构象的快速读谱辅助。 $p(r)$ 实空间距离分布；边界参数 $D_{\max}$。 IFT 间接傅里叶变换；程序入口 GNOM / GIFT。 多分散 尺寸分布抹平振荡；连接 Unified / Guinier–Porod 模型。

仪器、还原与样品

数据还原 2D→$I(q)$ 校正链总览。 绝对强度 $\mathrm{cm}^{-1}$ 尺度；标样常看 玻璃碳。 仪器模糊 分辨率卷积；狭缝数据常需 去模糊。 径向平均 各向同性一维化；配套 束阻 与 透射率。 SEC-SAXS 色谱联用减聚集；生物实验常见 辐射损伤 控制。 多重散射 厚样品/强散射体上的系统误差来源。

薄膜、聚合物与进阶方法

嵌段共聚物 连接 微相分离、ODT 与 $\chi$。 DWBA GISAXS 理论框架；读图常看 Yoneda 峰。 磁性 SANS 核/磁分离；进阶见 极化分析。 ab initio 形状 溶液低分辨重建；柔性体系看 EOM。 分形散射 幂律区与层次结构解读。 持续长度 半柔顺链；相关 关联长度。

建议学习顺序

  1. 先建立 $q$、衬度、$P(q)$ / $S(q)$ 四件套。
  2. 再学 Guinier 与 Porod，能对曲线「分区说话」。
  3. 做实验时补 数据还原、绝对强度、smearing。
  4. 按样品类型跳到 SAXS / SANS / GISAXS 等变体词条，再回 学习路径 或 专题手册。
