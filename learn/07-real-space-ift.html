07 · 实空间与 IFT

Real space & IFT — from $I(q)$ to $p(r)$, $D_{\max}$, and shape intuition

导语：Guinier 之后，为何还要进实空间

第 06 章从低 $q$ 得到 $R_g$ 与 $I(0)$，从高 $q$ 读界面渐近——两端锚点已立。但 Guinier 把整条可用曲线压成两个数；许多形貌（细长、多结构域、中空）在中等 $q$ 才显出特征，而单一 $R_g$ 无法区分「稍扁的球」与「略短的棒」。实空间的距离分布函数 $p(r)$ 把全曲线信息重排为「粒子内部任意两点间距的加权直方图」，直接给出最大尺寸 $D_{\max}$ 与形状直觉。

Svergun & Koch（2003）在生物大分子 SAS 综述中把 $p(r)$ 列为「超越整体参数」的关键一步：它利用全 $q$ 窗的信息，在 Guinier 窗偏窄或数据略噪时往往比单独 Guinier 更稳地给出 $R_g$ 与 $D_{\max}$，并为 ab initio 形状重建提供约束。但 $p(r)$ 的物理含义严格限于单粒子形式因子 $P(q)$ 的实空间表述——一旦 $S(q)\neq 1$，必须先拆相互作用（见下文 GIFT/SQ-IFT 节）。

本章是学习路径上的合成讲解：先弄清 $p(r)$ 与 $I(q)$ 的关系、为何不能直接傅里叶逆变换、如何走 GNOM / IFT 工作流、如何读 $p(r)$ 形状并与 Guinier 交叉验证，以及何时应放弃「单粒子 $p(r)$」叙事。正则化参数 $\alpha$、感知准则与文献细节见 IFT 专题；本章侧重「做完 Guinier 后下一步怎么做、结果怎么读」。

$p(r)$ 是什么：配对距离分布

在稀溶液、各向同性、单分散（或可近似为单一平均粒子）条件下，散射强度与实空间分布通过球对称傅里叶变换相联系：

$$I(q) = 4\pi\int_0^{D_{\max}} p(r)\,\frac{\sin(qr)}{qr}\,\mathrm{d}r$$

（记号随文献略有出入；$q$ 与生物文献的 $s$ 同义。）$p(r)$ 称为配对距离分布函数（pair-distance distribution function, PDDF）：$p(r)\,\mathrm{d}r$ 正比于粒子内距离落在 $[r,r+\mathrm{d}r]$ 的点对数目，并按衬度加权。在 GNOM 约定下，$p(r)$ 与 $P(q)$ 通过球对称傅里叶变换相联，且 $p(0)=0$、$p(r\ge D_{\max})=0$ 为硬边界。直观上：

  • $p(r)=0$ 当 $r=0$ 与 $r\ge D_{\max}$（边界条件）；中间有一个或多个峰。
  • $D_{\max}$： 粒子内最大两点间距——$p(r)$ 支撑的上界。它是实空间最重要的「尺寸」之一，比 $R_g$ 更直接对应几何延伸。
  • $p(r)$ 的峰位、不对称性、是否多峰，编码形状与结构域分隔，而不仅是整体大小。
$p(r)$ 与 $I(q)$ 的信息分工
  • $I(q)$： 实验直接量；低 $q$ 敏感整体尺寸与相互作用，高 $q$ 敏感界面与细部。
  • $p(r)$： 实空间重表述；把「振荡的形式因子」变成「可目视的距离直方图」——便于与分子模型、晶体结构投影比较。
  • 二者等价（理想数据下）： 知道完整无噪声 $I(q)$ 可唯一确定 $p(r)$；实践中 $q$ 有限、有噪声，反演必须正则化。

物理上，$p(r)$ 与粒子内散射长度密度涨落 $\Delta\rho(\mathbf{r})$ 的空间自相关有关：它统计的是「相距 $r$ 的两点同时落在高衬度区域」的加权概率，而非原子坐标本身。因此 $p(r)$ 对整体延伸、域间距离、中空/壳层等全局几何敏感，但对原子级细节不分辨——这与 Shannon 采样极限一致：可用 $q$ 窗越宽，$p(r)$ 上可辨的结构越多，但仍受 $D_{\max}$ 与信噪比封顶。

归一化约定因软件而异：ATSAS/GNOM 族常使 $p(r)$ 与 $I(0)$、$R_g$ 通过下节积分式自洽；阅读文献时务必核对「$p(r)$ 是否已按体积/电子数归一」。学习路径上只需记住：同一套数据、同一程序内比较 $p(r)$ 形状与 Guinier 参数即可，跨软件对比要查文档。

为何直接傅里叶变换失败：病态与 IFT

形式上，$p(r)$ 可由 $I(q)$ 的逆变换写出。对有限、带噪声的实验数据直接数值求逆，会得到剧烈振荡甚至大片负值的「解」——这是经典的病态（ill-posed）积分方程：核平滑、高频分量被压扁，噪声在实空间被放大（Glatter 1977）。

间接傅里叶变换（IFT） 的策略是：不直接逆变换，而是把 $p(r)$ 参数化为基函数展开（或分段表示），求解带 Tikhonov 正则化的最小化问题

$$T_\alpha[p] = \|I_{\mathrm{obs}}-I_{\mathrm{calc}}[p]\|^2 + \alpha\,\Omega[p],$$

其中第一项要求拟合 $I(q)$，第二项 $\Omega$ 惩罚粗糙（常取导数平方积分），$\alpha$ 为正则化强度：

  • $\alpha$ 过小 → 过拟合噪声，$p(r)$ 振荡、负值多。
  • $\alpha$ 过大 → 过光滑，丢失多峰与肩部，$I(q)$ 拟合变差。

GNOM（Svergun 等）在 ATSAS 管线中广泛使用：扫描 $\alpha$，用感知准则（光滑度、非负性、残差、稳定性等）自动选折中解。Svergun（1992）把人工「挑最好看的 $p(r)$」形式化为可计算的感知准则（perceptual criteria），并最大化综合评分选 $\alpha$——这是 GNOM 自动选参的理论基础。

准则含义（直觉）合格区间（量级）

学习路径上记住一句即可：IFT =「在拟合数据与光滑 $p(r)$ 之间找可辩护的折中」；$\alpha$ 过小对应 OSCILL 飙升、POSITV 下降；$\alpha$ 过大则 SYSDEV 偏离、主峰被抹平。细节与失效判据见 IFT 专题。

过拟合与欠光滑：如何自检

过拟合的典型信号不是「$\chi^2$ 很小」，而是解对 $\alpha$ 与数据裁切高度敏感：去掉最高 $q$ 的 10% 噪声点后，$p(r)$ 上突然出现大量细振荡；或 OSCILL $\gg 3$、POSITV 明显低于 1。欠光滑（$\alpha$ 过大）则相反：$I(q)$ 残差呈系统性弯曲（SYSDEV 偏低），$R_g^{\mathrm{IFT}}$ 与 Guinier 一致但 $D_{\max}$ 区内肩部被抹掉。合格报告应附带「$\alpha$ 扫描或相邻 $\alpha$ 两支 $p(r)$ 叠加图」，证明主峰稳定、细振荡未进入结论。

实践工作流：从 $D_{\max}$ 到可报告的 $p(r)$

  1. 前置检查： 背景已扣、$q$ 刻度可靠；最好已有 Guinier $R_g$、$I(0)$（第 06 章）。低 $q$ 至少若干点落在 Guinier 窗内，否则 $D_{\max}$ 难稳。
  2. 初选 $D_{\max}$： 从 $\sim 2.5$–$3\,R_g$（紧凑球常更接近 $D_{\max}\approx 2\sqrt{5/3}\,R_g$）试起；目标是使 $p(r)$ 在末端平滑衰减至零，而非在边界被「砍断」或拖出长噪声尾。
  3. 运行 GNOM / IFT： 输入 $I(q)$（及误差）；狭缝几何须纳入 smearing 核或先 desmear（针孔生物 SAXS 常可忽略）。
  4. 目视与数值验收：
    • $p(r)\ge 0$（允许末端微小数值噪声，但大片负值不可接受）；
    • 倒易空间拟合曲线穿过数据、残差无系统趋势；
    • IFT 导出的 $R_g$ 与 Guinier 偏差通常应 $\lesssim 5\%$–$10\%$。
  5. 扫描 $D_{\max}$： 略大、略小各试若干值；合格区间内 $p(r)$ 主峰形状应稳定，$R_g$ 变化小。选「末端自然归零且 OSCILL/负值最少」的一支。Svergun（1992）VALCEN 准则专门检查主峰是否落在 $D_{\max}$ 区间中部——边界截断会使 VALCEN 骤降。
  6. 报告： $D_{\max}$、$R_g$（IFT）、$I(0)$（若绝对尺度）、所用 $q$ 范围与软件/$\alpha$ 选取方式；生物发表另遵循 Trewhella 2017 推荐信息清单。

生物 SEC-SAXS 分帧后，每一帧的 $p(r)$ 应单独验证 Guinier 与 $D_{\max}$：Trewhella 2017 指出 SEC 峰展宽可能导致 UV 与 SAS 帧不对齐，浓度估计偏差会直接传递到 $I(0)/M_w$。学习路径上，SEC 数据仍走同一 IFT 流程，但只对主峰帧采信 $p(r)$。

算例：从 Guinier $R_g$ 估 $D_{\max}$ 初值

某蛋白稀溶液 Guinier 得 $R_g=3.2\,\mathrm{nm}$。先试 $D_{\max}=2.5\,R_g\approx 8\,\mathrm{nm}$；若 $p(r)$ 在 $r\approx 7$–$8\,\mathrm{nm}$ 被截成陡崖，增至 $D_{\max}=10$–$11\,\mathrm{nm}$ 直至末端平滑归零。均匀实心球应有 $D_{\max}\approx 2\sqrt{5/3}\,R_g\approx 8.3\,\mathrm{nm}$——与扫描结果主峰在 $D_{\max}/2$ 附近、IFT 与 Guinier $R_g$ 偏差 $\lesssim 5\%$ 自洽时，可采信该 $D_{\max}$。若无论怎么调 $D_{\max}$，$R_g^{\mathrm{IFT}}$ 始终比 Guinier 低 $>10\%$，优先查缓冲液扣除与低 $q$ 质量，而非继续增大 $D_{\max}$。

验收清单（学习路径版）
  1. $p(r)$ 在 $r\to D_{\max}$ 平滑 → 0，无强制截断尖角。
  2. 主峰区非负；若需强制非负约束，注明并与无约束解比较。
  3. 拟合 $I(q)$ 在低–中 $q$ 无系统欠/过拟合；高 $q$ 噪声区勿强追 $\chi^2\to 0$。
  4. $R_g^{\mathrm{IFT}}$ vs $R_g^{\mathrm{Guinier}}$：偏差 $>10\%$ → 先查本底、$q_{\min}$、$D_{\max}$，勿直接采信。
  5. 浓度系列：稀溶液极限下 $p(r)$ 形状应重叠；$I(0)/c$ 近似恒定。

读 $p(r)$ 形状：从球到中空

$p(r)$ 不是「真实原子坐标」，而是与数据相容的距离统计。下表给出稀溶液单分散情形下的典型目视印象（理想化；真实曲线常带多分散与噪声）：

球状 / 紧凑球状单峰、近似对称；峰近 $D_{\max}/2$$D_{\max}\approx 2\sqrt{5/3}\,R_g$（均匀球）最常见「教科书」形态
细长 / 椭球拉长峰偏左、右侧长拖尾$D_{\max}$ 远大于 $2 R_g$勿用球体公式反推半径

读图纪律：先问「是否满足单粒子稀溶液假设」，再谈形貌。宽化单峰 + 不稳的 $D_{\max}$ 更常提示多分散，而非「奇怪的形状」。Svergun & Koch（2003）强调：$p(r)$ 双峰在生物体系中既可能是结构域分隔，也可能是寡聚混合物——必须结合 SEC、Native PAGE 或浓度系列排除组分不纯，再解读峰位。

电荷体系与 $S(q)$ 干扰：何时 IFT 需「拆 $P$ 与 $S$」

标准 IFT 假设 $I(q)\propto P(q)$（或等价的单粒子 $p(r)$）。当$S(q)$ $\neq 1$ 时，测得强度实为 $I(q)\propto P(q)S(q)$，直接 IFT 得到的「$p(r)$」是被相互作用扭曲的平均核，低 $q$ 常被压低，稀溶液下才恢复的 $p(r)$ 形状才是单粒子 PDDF。

两条文献路线可处理浓相互作用体系，学习路径上知道分工即可：

  • GIFT（generalized IFT）： Fritz 等（2000）在 Glatter 族 IFT 上同时拟合模型无关 $p(r)$ 与带电胶体 $S(q)$（RMSA / HNC / RY 等闭包）。可在体积分数 $\lesssim 0.3$ 的带电胶体中分离粒内与粒间效应，但电荷、离子强度、半径、$\phi$ 存在简并——文献指出电荷与离子强度不能同时唯一确定，固定半径或独立电泳约束可破退化。
  • SQ-IFT： Fukasawa & Sato（2011）对 $S(q)$ 本身做 IFT，得到对关联函数 $g(r)$，可模型无关地读蛋白溶液中的有效结构因子与簇集行为；与 GIFT「先假设 $S(q)$ 模型」互补。

生物缓冲液中的蛋白在 $>5$–$10\,\mathrm{mg\,ml^{-1}}$ 常出现可测 $S(q)$；带电胶体（CTAB 胶束等）即使 1% 浓度也可能强相互作用。判断口诀：稀释后低 $q$ 抬升、$p(r)$ 形状变、$I(0)/c$ 非线性 → 先走浓度外推或 GIFT/SQ-IFT，勿把 $S(q)$ 压低当成「分子变紧凑」。第 08 章将显式拟合 $P(q)S(q)$；BioSAS 默认仍优先无限稀释。

由 $p(r)$ 得到 $I(0)$、$R_g$；绝对尺度提醒

在正确归一约定下，

$$I(0) = 4\pi\int_0^{D_{\max}} p(r)\,\mathrm{d}r,$$ $$R_g^2 = \frac{\displaystyle\int_0^{D_{\max}} r^2 p(r)\,\mathrm{d}r}{\displaystyle 2\int_0^{D_{\max}} p(r)\,\mathrm{d}r}.$$

（具体前因子随软件对 $p(r)$ 的定义略有差异；以所用程序文档为准。）要点：

  • $R_g$ 来自 $p(r)$： 利用全曲线，对 Guinier 窗略窄或略噪时往往更稳；仍须与第 06 章 Guinier 交叉。
  • $I(0)$ 与分子量： 仅当 $I(q)$ 已绝对标定（第 05 章）且浓度、衬度可靠时，$I(0)$ 才可换成 $M$。相对单位下，IFT 的 $I(0)$ 仍可用于同系列比较，但不可报化学分子量。
  • Porod 体积等衍生量： 生物 SAXS 常由 $p(r)$ / 不变量估体积；同样依赖绝对尺度与高 $q$ 质量。
与 Guinier 交叉的最低标准
  • 报告时同时给出 $R_g$（Guinier）与 $R_g$（$p(r)$）及各自不确定度。
  • 二者一致 → 增强对 $D_{\max}$ 与本底的信心；不一致 → 优先怀疑低 $q$ 质量、$D_{\max}$、聚集，而非「换一个更好看的 $\alpha$」。
  • 前向散射 $I(0)$：Guinier 外推与 $\int p(r)$ 应同量级一致。

Jeffries 等（2021）Primer 提醒：稀释极限 $S(q)\sim 1$ 是生物与许多胶体分析的默认前提，但必须用量纲一致的浓度系列验证，而非仅引用文献离子强度。Fukasawa（2011）在 lysozyme 与 HSA 体系显示，有效 $g(r)$ 可揭示静电簇集——这类情形下「单粒子 $p(r)$」叙事在浓样下失效，稀样或分离 $S(q)$ 后才可恢复 PDDF 解释。

一句话：IFT 假设「存在一个有限支撑的、光滑的、代表（平均）粒子的 $p(r)$」。假设崩塌时，继续调 $\alpha$ 只会得到更漂亮的假图——应换模型层级（第 08 章）或改实验。

常见坑

  • 坑 1：$D_{\max}$ 设太大 → 如何判断： $p(r)$ 末端长时间贴地振荡或出现负瓣；略减小 $D_{\max}$ 后主峰几乎不变而尾部变干净。逐级减小至末端自然归零。
  • 坑 2：$D_{\max}$ 设太小 → 如何判断： $p(r)$ 在右边界被砍成陡崖；拟合 $I(q)$ 低 $q$ 系统性偏差；$R_g^{\mathrm{IFT}}$ 偏小。加大 $D_{\max}$ 直至末端平滑。
  • 坑 3：把振荡 $p(r)$ 当「高分辨率结构」→ 如何判断： 略改 $\alpha$ 或裁掉高 $q$ 噪声后峰谷大变。那些细结构多半是噪声放大，不可写进结论。
  • 坑 4：忽视与 Guinier 的冲突 → 如何判断： $|R_g^{\mathrm{IFT}}-R_g^{\mathrm{Guinier}}|/R_g > 10\%$ 仍只贴 IFT 图。先查本底、聚集、$q_{\min}$ 与 $D_{\max}$。
  • 坑 5：浓溶液硬做单粒子 IFT → 如何判断： 稀释后 $p(r)$ 形状明显变；$I(0)/c$ 随 $c$ 降。先稀释或改用带 $S(q)$ 的模型（第 08 章）。
  • 坑 6：相对强度曲线谈「绝对」$I(0)$ / $M$ → 如何判断： 未做第 05 章标定却报分子量。相对单位只比较趋势；定量 $M$ 须绝对尺度 + 可靠浓度。
  • 坑 7：多分散混合物读成「多结构域」→ 如何判断： 双峰 $p(r)$ 在纯化/SEC 前后消失或峰比大变。先排除组分不纯，再谈结构域分隔。
  • 坑 8：狭缝 smearing 未入核 → 如何判断： Kratky/实验室狭缝数据上 $R_g$ 偏大、残差系统化。使用 smeared IFT 或先 desmear（第 03–04 章）。
  • 坑 9：浓盐缓冲液蛋白硬做 IFT → 如何判断： 蛋白–蛋白静电排斥使 $S(0)<1$，$p(r)$ 随浓度变、与晶体结构投影不符。Fukasawa（2011）在 HSA 体系显示离子强度改变有效 $g(r)$——先做盐/浓度系列或 SQ-IFT 读 $S(q)$，再谈单粒子 $p(r)$。
  • 坑 10：只报一张 $p(r)$ 无 $\alpha$ 信息 → 如何判断： Trewhella 2017 要求报告 $q$ 范围、$D_{\max}$ 选取、正则化方式及与 Guinier 交叉结果。缺任一即无法复现与评审。

小结

$p(r)$ 把 $I(q)$ 译成实空间距离语言：$D_{\max}$、形状直觉，以及与 Guinier 可比的 $R_g$、$I(0)$。直接逆变换病态，故用 IFT / GNOM 以正则化参数 $\alpha$ 在拟合与光滑之间折中；Svergun（1992）感知准则（OSCILL、POSITV、SYSDEV 等）为自动选 $\alpha$ 提供可辩护标准。合格 $p(r)$ 的标志是非负、末端归零、拟合无系统残差、与 Guinier 一致。多分散、强 $S(q)$（含带电胶体与浓蛋白液）、坏本底或错误 $D_{\max}$ 时，单粒子 IFT 叙事失效——浓相互作用体系用 GIFT 拆 $P$ 与 $S$，或 SQ-IFT 读 $g(r)$；下一章进入显式形式因子模型与多分散，把「一种 $p(r)$」升级为可辨识的形状与尺寸分布假设。
