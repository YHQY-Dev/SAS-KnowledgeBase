09 · 应用岔路

Application forks — bioSAS, soft matter, GISAXS

导语：基础读完，按样品几何分叉

第 01–08 章建立的是共用底座：什么是 SAS、$q$ 与衬度、仪器与 smearing、还原与绝对标定、Guinier/Porod、实空间 IFT、以及模型与多分散的可辨识性。到这里为止，你已能把一张校正后的 $I(q)$ 读成「尺寸—界面—形状」的粗粒度叙事。下一步不再是「再学一章通识」，而是按样品几何选岔路：

  • 溶液中的生物大分子（蛋白、核酸、复合物）→ 岔路 A · BioSAS
  • 体相软物质（聚合物溶液/熔体、胶束、胶体、凝胶、两相体系）→ 岔路 B · Soft matter
  • 薄膜 / 界面 / 有序纳米结构表面→ 岔路 C · GISAXS

三条路共享同一套散射物理，但第一批要回答的问题、默认假设、必做对照实验完全不同。选错岔路，最常见的代价不是「公式不会用」，而是把聚集当成柔性、把 $S(q)$ 当成 $P(q)$、或把掠入射干涉条纹当成透射形式因子振荡。本章是导流图：每条岔路给出何时走、先读哪几个专题、典型科学问题与下一跳链接；最后用决策表与常见坑帮你自检。

岔路信号：三条「报警灯」

在翻专题手册之前，用三个快问定位岔路（Jeffries 2021；Grillo 2008；Müller-Buschbaum 2009）：

  • 样品是各向同性稀溶液、问单分子/单粒子？ → 先看 BioSAS 或软物质稀溶液叙事；几何为透射，$S(q)\approx 1$ 是默认待证假设。
  • 体相浓度高、需操控 H/D 衬度或读 $S(q)$ 峰？ → 软物质岔路；SANS 对比变分几乎是标配，聚合物/胶体/胶束走此路。
  • 样品是薄膜/涂层/衬底上的纳米结构、需要面内周期或埋藏界面？ → GISAXS；透射 Guinier 公式不适用，必须进入 DWBA 与二维 $q$ 分解。

岔路 A · BioSAS（溶液生物分子）

何时走这条岔路

样品是可纯化的生物大分子稀溶液（蛋白、核酸、复合物、部分膜蛋白–去垢剂复合物），科学问题围绕寡聚态、整体尺寸、柔性/结构域、配体结合引起的构象变化。Trewhella 等（2017）强调：结构建模级 BioSAS 的前提是可证明的单分散稀溶液形式因子——任何聚集、缓冲不匹配或 SEC 峰不纯都会直接否定 $p(r)$ 与 $M_w$ 解释。

典型问题 → 工具映射

  1. 分子量 / 寡聚态： 绝对标定 后 $I(0)$ 是否与序列 $M_w$、已知寡聚体一致？浓度升高是否聚集？
  2. 整体尺寸： Guinier $R_g$ 是否稳定？与 $p(r)$ 给出的 $R_g$、$D_{\max}$ 是否交叉一致？
  3. 形状与柔性： $p(r)$ 是球状单峰、细长双峰，还是柔性链特征？Kratky 图是否提示无序？
  4. 复合物与构象变化： 配体结合、突变、缓冲条件是否改变 $R_g$/$D_{\max}$，而非仅改变本底？
  5. 高阶建模： 刚性体/系综拟合（Svergun & Koch 2003）——仅在上述验证通过后进入。

先读专题（推荐顺序）

  1. 绝对标定 + 第 05 章 —— $I(0)/c$ 与 $M_w$ 的基石
  2. BioSAS —— 缓冲匹配、浓度系列、Kratky/Invariant 验证
  3. IFT / $p(r)$ —— $D_{\max}$、形状直觉（第 07 章实操）
  4. ATSAS —— GNOM、CRYSOL、刚性体/系综管线

下一跳：何时离开 BioSAS 岔路

  • 样品实为胶束/高浓度胶体 → 岔路 B（$S(q)$、多分散）
  • IDP/大柔性系综 → 系综方法，勿把宽 $p(r)$ 当刚性包络
  • 膜蛋白 SEC 峰含空胶束 → SEC-SAS 分帧；见 BioSAS 专题
BioSAS 上路清单
  1. 完成浓度系列（至少 3 点）并外推至无限稀释或确认 $S(q)$ 可忽略。
  2. 缓冲精确匹配；检查低 $q$ 上翘（聚集/气泡）与高 $q$ 本底。
  3. 报告 Guinier $R_g$、$I(0)$，并与 IFT $p(r)$ 交叉（见第 06–07 章）。
  4. 有绝对标定时，用 $I(0)/c$ 核对 $M_w$；无标定时只谈相对比较。
  5. 进入专题：BioSAS、IFT / $p(r)$、绝对标定。

BioSAS 把第 05–07 章用到极致：没有可靠的绝对尺度与缓冲扣除，后面的寡聚态与柔性讨论会建在沙子上。Trewhella 2017 要求报告样品来源、序列/FASTA、缓冲组成、浓度测定方法、$q$ 范围与 Guinier/$p(r)$ 交叉结果——学习路径上把这些当作发表级自检表。若你的样品其实是胶束、聚合物或高浓度胶体，请改走岔路 B，不要硬套生物发表流程。

岔路 B · Soft matter / 聚合物 / 胶体

何时走这条岔路

样品为聚合物溶液/熔体、胶束、纳米粒子分散体、凝胶、嵌段共聚物体相或浓溶液，核心科学问题常是：哪一相在散射、相互作用多强、尺寸分布多宽、是否存在层级结构（初级粒子 → 聚集体 → 网络）。Grillo（2008）指出软物质 SANS 的「主战场」是衬度操控与$S(q)$，而非单一分子的 $D_{\max}$。Hamley（2004）与 Trask（2017）的嵌段共聚物综述进一步说明：熔体微相、溶液胶束与结晶态需不同 $P(q)$/$S(q)$ 叙事——体相 BCP 常需 RPA 结构因子，溶液胶束则球/柱 $P(q)$ + 胶束间 $S(q)$。

典型问题 → 工具映射

  • 衬度从何而来？溶剂、氘化、组分匹配能否分离某一相？
  • 形式因子 $P(q)$ 描述单粒子/单链，还是已被结构因子 $S(q)$ 调制？稀溶液近似是否成立？
  • 多分散（尺寸、形状、电荷）是否淹没振荡？表观 $R_g$ 是否被大端加权？
  • 是否存在层级结构（初级粒子 → 聚集体 → 网络）需要 Beaucage / Guinier–Porod 类粗粒度描述？

先读专题（推荐顺序）

  1. 第 08 章 + SANS 衬度与对比变分 —— 写清 $\Delta\rho$、匹配点、背景
  2. 数据还原、绝对标定 —— 体相/浓样仍须绝对强度
  3. 聚合物与嵌段共聚物 —— 熔体微相、胶束、BCP 溶液
  4. 第 06 章 Guinier/Porod + 第 08 章 Beaucage PDI —— 宽 $q$ 粗读与多分散指数

下一跳：何时升级到 GISAXS 或 BioSAS

  • 嵌段共聚物 薄膜、定向自组装涂层 → 岔路 C（面内周期、Yoneda 峰）
  • 稀溶液中单一胶束/复合物问「化学计量寡聚」→ 若可纯化且 $S(q)\approx 1$，可借 BioSAS 验证流程
  • 带电胶体 $S(q)$ 强 → 第 08 章 GIFT/SQ-IFT 思路或盐/$\phi$ 系列（第 07 章电荷干扰节）

在透射几何下，稀体系近似写为

$$I(q) \propto n\,V^2\,(\Delta\rho)^2\,P(q)\,S(q),$$

其中 $S(q)\to 1$ 仅在相互作用可忽略时成立。聚合物溶液、带电胶体、浓缩胶束常常必须显式处理 $S(q)$，或用稀释/盐浓度系列把相互作用「关掉」。

Soft matter 上路清单
  1. 先写清：你测的是哪一相的衬度？预期 $\Delta\rho$ 数量级是否合理？
  2. 做浓度（或体积分数）系列：低 $q$ 随浓度系统性压低/抬高 → 优先怀疑 $S(q)$。
  3. 多分散宽时，少报「精确几何半径」，多报分布矩与模型假设。
  4. 还原与绝对尺度仍是前提：见 数据还原、绝对标定；Guinier/Porod 粗读见第 06 章。
  5. 还原与绝对尺度仍是前提：见 数据还原、绝对标定；Guinier/Porod 粗读见第 06 章。
  6. 嵌段共聚物、胶束形状：深入 聚合物/BCP 专题；SANS 实验细节对照 Grillo（2008）软物质教程。

岔路 C · GISAXS / 表面与薄膜

何时走这条岔路

科学问题需要面内纳米周期、薄膜厚度/粗糙度、埋藏界面或衬底上自组装结构，且样品为衬底支撑的薄膜（嵌段共聚物薄膜、纳米点阵、LB 膜、涂层等）。Müller-Buschbaum（2009）与 Hexemer & Müller-Buschbaum（2015）强调：GISAXS 的价值在于统计平均、可探测埋藏结构、可用 $\alpha_i$ 调穿透深度——这些透射 SAS 无法替代。

典型问题 → 工具映射

  • 面内周期 $d$： 侧向 Bragg 峰位 $q_y \approx 2\pi/d$；勿与 $P(q)$ 振荡极小混淆
  • 形貌与高度相关： 镜面杆、Yoneda 翼、条纹间距 —— 需 DWBA 模拟
  • 深度分辨： $\alpha_i$ 接近 $\alpha_c$ 增强表面；增大 $\alpha_i$ 探埋藏界面（Hexemer 2015）
  • 动力学： 退火、溶剂退火、原位加热 —— 多 $\alpha_i$ 时间序列

GISAXS 不是「把样品换成薄膜后的普通 SAXS」。掠入射下，X 射线在表面发生反射与折射，散射振幅是入射场 + 反射场与样品起伏/埋藏结构耦合的结果；常用语言是 DWBA：把动力学反射作为参考波，对侧向结构做微扰散射。实践直觉：

若你的样品是溶液蛋白或体相胶体，却只因为「想看表面」就改 GISAXS 而不改问题定义，通常会得到难以解释的干涉条纹。正确入口是专题 GISAXS，并回顾第 03 章的掠入射几何。

先读专题（推荐顺序）

  1. 第 03 章仪器 + GISAXS 专题 —— $q_x,q_y,q_z$、临界角、二维图识读
  2. DWBA 与 Yoneda 峰 —— 先认特征再拟周期
  3. 软物质薄膜应用：Hexemer & Müller-Buschbaum（2015）；BCP 薄膜见 聚合物/BCP

下一跳：与透射 SAS 的衔接

  • 同一 BCP 样品：体相稀溶液透射 → 岔路 B；旋涂薄膜 → 岔路 C
  • 纳米粒子溶液问粒径 → 岔路 B；同一粒子自组装单层 → 岔路 C
GISAXS 上路清单
  1. 确认科学问题真的需要面内/深度分辨信息，而不是透射体相平均。
  2. 记录并报告 $\alpha_i$、样品对准、临界角估计；同一结构至少测 2–3 个入射角。
  3. 先识别 Yoneda / 镜面特征，再谈侧向周期拟合。
  4. 不要把 GISAXS 强度直接套用透射 Guinier 公式报「溶液 $R_g$」。
  5. 深入阅读：GISAXS 专题。

跨岔路仍必须保留的技能

分叉不等于抛弃基础。无论走 A/B/C，下列能力仍然是「可发表」的门槛（Jeffries 2021；Trewhella 2017）：

  • 数据还原： 掩膜、归一化、背景、误差条诚实（第 04 章 / 还原专题）。
  • 绝对尺度（能做就做）： 跨样品、跨束线比较与 $M_w$/体积分数都依赖它（第 05 章 / 标定专题）。
  • Guinier 理智检查： 即使最终用复杂模型，低 $q$ 的 $R_g$ 与 $I(0)$ 仍是最快的异常警报（第 06 章 / Guinier 分析）。
  • 假设显式化： 写清 $S(q)\!\approx\!1$、单分散、锐利界面、透射几何等——哪一条不成立，就要改模型或改实验。
  • 误差与 $q$ 范围诚实： 三条岔路均须报告 $q_{\min},q_{\max}$ 及误差来源；GISAXS 另须 $\alpha_i$ 与曝光条件。

混合样品：何时需要两条岔路

真实课题常跨岔路。常见组合：

常见坑：选错岔路时会发生什么

  • 坑 1：几何假设错误 → 如何判断： 把薄膜 GISAXS 数据当透射溶液 SAS 方位平均；或把 Kratky 狭缝数据当针孔数据做 Guinier。先核对第 03 章几何与 smearing。
  • 坑 2：忽略相互作用 → 如何判断： 高浓度蛋白/胶体低 $q$ 被压低，表观 $R_g$、$I(0)$ 偏小，却报告「分子变紧凑」。做稀释系列；显式引入 $S(q)$。
  • 坑 3：BioSAS 流程硬套胶束/聚合物 → 如何判断： $p(r)$ 不稳定、$D_{\max}$ 随正则化乱跳、分子量对不上。改走衬度 + 多分散 + $S(q)$ 叙事。
  • 坑 4：把 GISAXS 当透射 SAS → 如何判断： 用透射 Guinier 公式从 $q_y$ 切线「报 $R_g$」；忽略 Yoneda 与 $\alpha_i$ 依赖。回到 DWBA 框架与 GISAXS 专题。
  • 坑 5：衬度叙事缺失 → 如何判断： 多组分体系拟合出「漂亮尺寸」，却说不清散射来自哪一相。先算/测 $\Delta\rho$，再谈模型。
  • 坑 6：跳过还原与标定直冲高级拟合 → 如何判断： 参数随本底、掩膜、绝对因子剧烈漂移。回到第 04–05 章把地基浇实。
  • 坑 7：用错「下一章」 → 如何判断： 需要面内周期却去钻溶液 IFT；需要寡聚态却去拟合薄膜 DWBA。用上表重选岔路。

小结

学习路径正文到此收束：01–08 是共用工具箱，09 是应用导流图。按样品几何选择 BioSAS、软物质或 GISAXS——用本章「岔路信号」三问与决策表锁定下一读与第一批实验。BioSAS 依赖绝对标定与单分散验证（Trewhella 2017）；软物质依赖衬度与 $S(q)$ 拆解（Grillo 2008；Hamley 2004）；GISAXS 依赖 DWBA 与多 $\alpha_i$ 二维识读（Müller-Buschbaum 2009）。始终保留还原、绝对尺度与 Guinier 自检。后续深入请进入专题手册按题阅读，术语与符号以词条库为准；需要总览时返回学习路径总览。
