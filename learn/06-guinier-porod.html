06 · Guinier 与 Porod

Guinier & Porod — $R_g$, $I(0)$, and high-$q$ interface asymptotes

导语：先读两端，再谈中间

校正并（最好）绝对标定之后的 $I(q)$，不必立刻做全曲线形状拟合。多数工作流先回答两个「端点问题」：

  • 低 $q$（Guinier 区）： 粒子/分子有多大？前向强度 $I(0)$ 是否合理？
  • 高 $q$（Porod 区）： 界面是否尖锐？比表面积量级如何？

两端读通后，中间区（形式因子振荡、结构因子峰）才有可靠语境。本章给出 Guinier / Porod 的公式直觉、适用条件、作图与拟合流程、Hammouda 统一 Guinier–Porod 模型，以及一组常见坑。实空间完整反演见第 07 章与 IFT 专题。

回转半径 $R_g$ 是什么

回转半径 $R_g$ 是散射长度密度「质心」的二阶矩：

$$R_g^2 = \frac{\displaystyle\int (\mathbf{r}-\mathbf{r}_0)^2\,\Delta\rho(\mathbf{r})\,\mathrm{d}V}{\displaystyle\int \Delta\rho(\mathbf{r})\,\mathrm{d}V}.$$

它不是「半径」本身，而是形状的整体尺寸度量。对均匀球体 $R = R_g\sqrt{5/3}$；对细长棒、薄盘、空心壳，换算系数不同。稀溶液单分散无相互作用体系中，$R_g$ 几乎总是第一批要报告的参数——它对绝对强度标定不敏感（Guinier 斜率只依赖形状），但对低 $q$ 数据质量极度敏感。

对象$R_g$ 与几何尺寸备注
均匀实心球（半径 $R$）$R_g = R\sqrt{5/3}\approx 0.775\,R$最常用换算
细长棒（长度 $L$，忽略截面）$R_g \approx L/\sqrt{12}$低 $q$ 看到长度主导
薄圆盘（半径 $R$）$R_g \approx R/\sqrt{2}$厚度贡献可另加
高斯链（均方末端距 $\langle R^2\rangle$）$R_g = \sqrt{\langle R^2\rangle/6}$聚合物溶液经典关系

Guinier 近似

在 $q R_g \ll 1$ 时，任意形状（各向同性平均后）的散射可展开为

$$I(q) \approx I(0)\,\exp\!\left(-\frac{q^2 R_g^2}{3}\right).$$

两边取对数得直线形式（Guinier 图）：

$$\ln I(q) = \ln I(0) - \frac{R_g^2}{3}\,q^2.$$

以 $q^2$ 为横轴、$\ln I$ 为纵轴：斜率 $= -R_g^2/3$，截距 $= \ln I(0)$。这就是Guinier 分析的操作定义。

适用窗口：$q R_g$ 上限

教科书常写 $q R_g < 1$。实践中上限取决于形状与多分散（Glatter & Kratky 1982；Hammouda 2010）：

散射体类型推荐 $q_{\max} R_g$偏离症状备注
紧凑球、近球形蛋白$\lesssim 1.3$$>1.3$ 时 $\ln I$–$q^2$ 下弯，$R_g$ 高估Hammouda 球形 micelle：$R_g=46$ Å，拟合至 $Q=0.075$ Å$^{-1}$，$Q_1=0.053$ Å$^{-1}$
细长棒、盘状、中空壳$\lesssim 0.8$–$1.0$中间区 $1/q$ 或 $1/q^2$ 提前出现须用广义 Guinier（$s=1$ 棒，$s=2$ 盘）

多分散体系收紧 $q_{\max}$；保证 $q_{\min} R_{g,\max} \ll 1$表观 $R_g$ 偏大（大粒子加权）见下文多分散节
浓溶液（$S(q)\neq 1$）不适用经典 Guinier低 $q$ 压低，$I(0)$、$R_g$ 偏小稀释或显式 $S(q)$

自洽流程：先粗估 $R_g$ → 检查 $q_{\max} R_g$ → 越界则缩短区间 → 迭代至 $\mathrm{d}\ln I/\mathrm{d}q^2$ 在拟用窗内近似常数。ATSAS Autorg 自动搜索线性段，但生物 SAXS 仍建议目视 $\ln I$–$q^2$ 图。

Guinier 拟合检查清单
  1. 确认低 $q$ 无束阻挡、无寄生散射抬高；$\mathrm{d}\ln I/\mathrm{d}q^2$ 在拟用区间近似常数。
  2. 用 $q_{\max} R_g \approx 1.0$–$1.3$ 自洽：先估 $R_g$，再检查所用 $q_{\max}$ 是否越界；越界则缩短区间重拟合。
  3. 报告 $R_g$、$I(0)$ 及拟合区间；生物 SAXS 另报 Autorg / 手工拟合对比。
  4. 与 $p(r)$ 得到的 $R_g$（第 07 章）交叉：偏差 $>10\%$ 须查聚集、本底或 $q_{\min}$。
  5. 绝对标定后，$I(0)/c$ 应与化学分子量粗略一致（第 05 章）。

$I(0)$ 的物理含义

前向散射 $I(0)$ 正比于「总衬度体积」的平方：稀溶液单分散时

$$I(0) = n\,V^2\,(\Delta\rho)^2 = c\,M\,(\Delta\rho_M)^2 / N_A$$

（记号随质量浓度 / 数密度约定而变）。因此：

  • $I(0)$ 对聚集极敏感：二聚体可使 $I(0)$ 近似翻倍（同浓度）。
  • $I(0)$ 对浓度误差、绝对标定误差线性敏感。
  • 仅相对强度时，$I(0)$ 仍可用于同系列相对比较，但不可报分子量。

手工示例（数量级）

设某球状蛋白 Guinier 区测得斜率 $s = \mathrm{d}\ln I/\mathrm{d}q^2 = -120\,\mathrm{\AA}^2$（$q$ 以 Å$^{-1}$ 计）：

$$R_g = \sqrt{-3s} = \sqrt{360}\approx 19.0\,\mathrm{\AA}.$$

若拟合用到 $q_{\max}=0.06\,\mathrm{\AA}^{-1}$，则 $q_{\max} R_g \approx 1.14$，落在可接受窗口；若盲目用到 $0.10\,\mathrm{\AA}^{-1}$，$q R_g\approx 1.9$，直线已弯，结果不可信。

算例 2（绝对 $I(0)$ 与分子量）： 绝对标定后 Guinier 截距得 $I(0)=0.12\,\mathrm{cm}^{-1}$，浓度 $C=5\,\mathrm{mg/mL}=5\times 10^{-3}\,\mathrm{g/cm}^3$，$\Delta\rho_M = 2.0\times 10^{10}\,\mathrm{cm}^{-2}$（电子密度差）：

$$M = \frac{I(0)\,N_A}{C\,(\Delta\rho_M)^2} \approx \frac{0.12 \times 6.02\times 10^{23}}{5\times 10^{-3} \times (2.0\times 10^{10})^2} \approx 3.6\times 10^4\,\mathrm{g/mol}.$$

与化学 $M$ 偏差 $>15\%$ 时优先查标定、浓度、聚集，而非强行扩 Guinier 区间。

多分散对 Guinier 的影响

多分散时 $I(q)$ 为尺寸分布加权平均，Guinier 斜率给出的是表观 $R_g$（接近 $z$-平均而非数平均）。分布宽时：

  • $\ln I$–$q^2$ 线性变差，$R_g$ 对大尺寸端敏感；$q_{\min}$ 须满足 $q_{\min} \bar{R}_g \ll 1$ 且覆盖足够多小粒子贡献。
  • 粒径分布半宽 $>30\%$ 时，经典 Guinier 常与 $p(r)$ 的 $R_g$ 偏差 $>10\%$——应以 IFT 交叉验证（第 07 章）。
  • 不可将表观 $R_g$ 直接代入 $R=\sqrt{3/5}\,R_g$ 球半径换算；须已知或假设分布形状。

Porod 定律与高 $q$ 渐近

当 $q$ 足够高、探针尺度远小于粒子尺寸、界面近似锐利时，散射强度服从Porod 定律：

$$I(q)\;\xrightarrow{q\to\infty}\; \frac{2\pi\,(\Delta\rho)^2\,S}{q^4},$$

其中 $S$ 为样品中（单位体积）的界面面积。对数–对数图上斜率约 $-4$。实务上常作 $q^4 I(q)$ vs $q$（或 vs $q^4$）检验是否趋近平台。绝对强度下，两相体系比表面积：

$$S_V = \frac{1}{2\pi (\Delta\rho)^2}\lim_{q\to\infty} q^4 I(q)$$

（$S_V$ 为单位体积界面面积；须已知 衬度 $\Delta\rho$ 与可靠背景扣除）。Hammouda（2010）强调：$d=4$ 为光滑粒子表面；$d=3$ 可为粗糙界面或不良溶剂 collapsed 链；$d=5/3$ 为良溶剂 fully swollen 链；$d=2$ 为高斯链或薄片；$d=1$ 为刚性棒——须结合 $q$ 位置判断，勿把聚合物质量分形当界面 Porod。

指数偏离：粗糙与弥散界面

高 $q$ 行为典型含义注意
$I\sim q^{-4}$锐利、光滑界面（经典 Porod）背景扣除后才可信
$I\sim q^{-3}$–$q^{-4}$ 之间分形表面或粗糙界面需足够宽的高 $q$ 窗口
$I\sim q^{-2}$ 等薄片、高斯链等「质量分形」区未必是「坏 Porod」
指数衰减更快弥散界面（Debye–Bueche 等）勿硬套 $q^{-4}$ 求面积

Porod 不变量

对两相体系，不变量

$$Q = \int_0^\infty q^2 I(q)\,\mathrm{d}q = 2\pi^2\,(\Delta\rho)^2\,\phi(1-\phi)$$

（绝对强度、$q$ 覆盖足够时）联系体积分数 $\phi$。实践中积分上下限截断会产生系统误差：低 $q$ 用 Guinier 外推，高 $q$ 用 Porod 尾外推——这正是绝对标定与背景质量再次关键的地方。

（绝对强度、$q$ 覆盖足够时）联系体积分数 $\phi$。实践中积分上下限截断会产生系统误差：低 $q$ 用 Guinier 外推，高 $q$ 用 Porod 尾外推——这正是绝对标定与背景质量再次关键的地方。

Porod 平台算例（数量级）

绝对标定后，高 $q$ 区 $q^4 I(q)$ 平台 $\approx 8\times 10^{-5}\,\mathrm{cm}^{-1}\,\mathrm{Å}^4$，$\Delta\rho = 3\times 10^{-6}\,\mathrm{Å}^{-2}$：

$$S_V \approx \frac{8\times 10^{-5}}{2\pi (3\times 10^{-6})^2} \approx 1.4\times 10^{6}\,\mathrm{Å}^{-1} \approx 140\,\mathrm{m}^2/\mathrm{cm}^3.$$

若 $\sigma/I > 30\%$ 或平台来自常数背景而非 $q^{-4}$，此数无物理意义——先查第 04 章本底与第 05 章标定。

Hammouda Guinier–Porod 统一模型

许多软物质、纳米粒子体系同时需要低 $q$ 尺寸与高 $q$ 幂律。Hammouda（2010）给出在 Guinier 区与 Porod 区连续衔接的经验模型：低 $q$ 用 Guinier 形式，高 $q$ 用 $q^{-d}$，并在衔接点 $Q_1$ 强制函数值与一阶导数连续。

球形（$s=0$）：

$$I(Q) = G\exp\!\left(-\frac{Q^2 R_g^2}{3}\right)\quad (Q \leq Q_1), \qquad I(Q) = D/Q^d\quad (Q \geq Q_1),$$ $$Q_1 = \frac{1}{R_g}\sqrt{\frac{3d}{2}}, \qquad D = G\exp\!\left(-\frac{d}{2}\right)\left(\frac{3d}{2}\right)^{d/2}\frac{1}{R_g^d}.$$

非球形推广（$s=0$ 球，$s=1$ 棒，$s=2$ 盘）：

$$I(Q) = \frac{G}{Q^s}\exp\!\left(-\frac{Q^2 R_g^2}{3-s}\right)\quad (Q \leq Q_1), \qquad Q_1 = \frac{1}{R_g}\sqrt{\frac{(d-s)(3-s)}{2}},$$

约束 $d > s$ 且 $s < 3$。圆柱等可含两个 Guinier 区（低 $q$ 见全长 $R_{g2}$，中间 $q$ 见截面 $R_{g1}$）加一个 Porod 区——共三或四个 $q$ 区（Hammouda §4）。

适用场景：宽 $q$ 范围的粗粒度描述、多层级结构的第一轮参数化（如 Pluronic P85 从 unimer→球状 micelle→柱状 micelle→层状 micelle，Hammouda Table 1）。不适用：需要精确 $P(q)$ 振荡、强 $S(q)$ 峰、或必须报生物发表级 $R_g$/$D_{\max}$ 时——后者仍优先经典 Guinier + IFT。模型不能再现单分散紧凑粒子的振荡；峰可用 $P(q)\times S(q)$ 扩展。

与 Beaucage 统一拟合的关系

Beaucage 全局散射函数（含多分散尺寸分布分析，2004 等）把 Guinier 与幂律写成可叠加的多层级形式，广泛用于气溶胶、枝状聚合物、层级纳米结构。与 Hammouda 模型同属「粗粒度统一描述」家族：优点是参数少、覆盖全 $q$；代价是参数相关、物理解释需额外约束。第 08 章讨论模型可辨识性时会再回到这一点。

工作流：从曲线到可报告数字

  1. 目视分区： 对数–对数图标出疑似 Guinier 平坦/缓降区与高 $q$ 幂律区。
  2. Guinier： 作 $\ln I$–$q^2$ 图；选线性段；自洽 $q_{\max}R_g$；记录 $R_g$、$I(0)$。
  3. Porod： 作 $q^4 I(q)$；判断平台；必要时估 $S$（须绝对强度 + 已知 $\Delta\rho$）。
  4. 交叉验证： $R_g$（Guinier）vs $R_g$（$p(r)$）；$I(0)$ vs 分子量预期；高 $q$ 斜率是否与形貌假设一致。
  5. 再升级： 需要 $D_{\max}$、$p(r)$ 形状 → 第 07 章 IFT；需要显式形状/多分散 → 第 08 章。

常见坑

  • 坑 1：Guinier 区间选太宽 → 如何判断： $\ln I$–$q^2$ 已明显下弯仍硬拟合；$q_{\max}R_g>1.5$。收紧 $q_{\max}$ 或换形状感知方法。
  • 坑 2：低 $q$ 上翘当「大 $R_g$」→ 如何判断： 聚集、气泡、束阻挡、本底欠扣都会抬高最低几个点。先查还原与样品，勿盲目外推 $I(0)$。
  • 坑 3：狭缝 smearing 未处理 → 如何判断： Kratky / 某些 USAXS 几何下 Guinier 区被抹平，$R_g$ 偏大。desmear 或拟合 smeared 模型（第 03–04 章）。
  • 坑 4：Porod 区被常数背景淹没 → 如何判断： 高 $q$ 平台来自荧光/非相干/探测器本底，假 $q^{-4}$。先可靠扣本底再谈界面面积。
  • 坑 5：多分散当单分散 Guinier → 如何判断： 表观 $R_g$ 被大端加权；分布宽时 Guinier 线性差。报告时注明「$z$-平均类」并考虑尺寸分布模型。
  • 坑 6：结构因子当形式因子 → 如何判断： 浓溶液低 $q$ 被 $S(q)$ 压低，表观 $R_g$、$I(0)$ 偏小。稀释至相互作用可忽略，或显式建模 $S(q)$。
  • 坑 7：相对强度硬算 Porod 面积 → 如何判断： 缺绝对标定与 $\Delta\rho$ 时，$S$ 只有任意单位。先完成第 05 章。

小结

Guinier 从低 $q$ 给出 $R_g$ 与 $I(0)$——整体尺寸与「有多少散射物质」的第一估计；Porod 从高 $q$ 约束界面锐利度与比表面积。二者是读 $I(q)$ 的两端锚点。下一章进入实空间：用 IFT 从整条（可用）曲线得到 $p(r)$ 与 $D_{\max}$，并与本章 $R_g$ 交叉验证。
